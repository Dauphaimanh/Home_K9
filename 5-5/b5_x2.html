To set random colors :
1, Add random-color class.
2, To change color, open theme{num}.css, find "random-colors".
3, You can change color from here.
- The assigned colour seems to depend on the object name. So conceivably one could write a script to affect the chosen colour by varying the object name; appending a number, for example. That seems like overkill to me though, and not very elegant.

To use theme color :
- On the Design tab, click the arrow under Variants, point to Colors, and then click the name of the theme color you created.
To add new theme color :
1, On the Design tab, click the arrow under Variants, and point to Colors.
2, Click Customize Colors.
3, Click a color box that you want to change. For example, Accent 1.
3, Click a new color in the Colors dialog box.

To change icon :
1, Tap and hold an empty are of the home screen and select Themes.
2, Tap icons and select an icon pack..
3, Tap Download (if free) or the price (if paid) and follow the on-screen instructions to confirm.
